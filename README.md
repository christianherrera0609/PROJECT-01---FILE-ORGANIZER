# PROJECT 01 - FILE ORGANIZER

## Objective
Build a script to automate file cleaning using folder location and file extension.

## Tools
- Python
- Linux
- Windows

## Summary
Script mini-project that takes a folder location and file extension as input, collects matching files and moves them to a destination folder. 
Handles incorrect extensions, and creates the destination folder if it does not exist.

## What I Learned
- Libraries: `os`, `shutil`, `re`, `time`
- Regex extension
