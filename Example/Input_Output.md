# Input / Output Analysis

## Input
The script requires the exact path of the folder you want to clean, the destination folder and file extension.
when you type the extension, use only the characters after the last dot. type "txt" for ".txt" or "png" for ".png" files.
![alt text](image.png) (Example/Images/image-1.png)

## Output
The script output gives you the number of files moved and the time it took to complete the process. This is useful for future analysis.
